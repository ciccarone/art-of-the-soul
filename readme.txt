## Description

Architecture of the Soul is a WordPress theme developed for a website/book that explores the relationship between architecture and the human soul.

## Installation

1. Download the theme from the provided location.
2. In your WordPress admin panel, go to `Appearance > Themes` and click the `Add New` button.
3. Click `Upload Theme`, select the theme file you've just downloaded and then click `Install Now`.
4. Once installed, click `Activate` to start using the theme.

## Features

- **Responsive Design:** The theme is designed to look great on both desktop and mobile devices.
- **Custom Post Types:** Includes custom post types for architectural projects and philosophical discussions.
- **Customizable Colors & Fonts:** Easily change the theme colors and fonts to match your branding.
- **SEO Friendly:** Built with best SEO practices in mind.

## Customization

Architecture of the Soul theme supports WordPress Customizer, allowing you to preview changes in real time. In your WordPress admin panel, navigate to `Appearance > Customize` to make changes to the theme.
